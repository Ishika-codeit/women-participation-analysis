# 👩 Women’s Participation in Parliament Analysis using R

## 📌 Project Overview

Women's representation in parliament is a key indicator of gender equality, political inclusion, and democratic development. This project explores the factors associated with women's participation in parliamentary institutions through data analysis, visualization, and machine learning techniques using **R Programming**.

The project focuses on understanding trends, identifying important variables, and building an interpretable **Decision Tree Model** to analyze patterns in parliamentary representation. Through exploratory data analysis and statistical visualization, the project provides insights into the state of women's political participation and the factors that may influence it.

---

## 🎯 Objectives

- Analyze women's participation in parliament using real-world data.
- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Identify patterns, trends, and relationships within the dataset.
- Visualize important findings using graphs and charts.
- Build a Decision Tree model for predictive analysis.
- Interpret model outputs and derive meaningful insights.

## 🚀 How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/women-parliament-analysis.git
```

### Step 2: Open the Project in RStudio

Open the project folder and load the analysis script.

### Step 3: Install Required Packages

```R
install.packages(c(
  "tidyverse",
  "dplyr",
  "ggplot2",
  "rpart",
  "rpart.plot"
))
```

### Step 4: Execute the Script

```R
source("analysis.R")
```
