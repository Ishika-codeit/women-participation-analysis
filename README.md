**📊 Women's Participation in Parliament Analysis**
**📌 Project Overview
**
This project analyzes women's representation and participation in parliament using statistical and machine learning techniques in R. The objective is to explore trends, identify key factors influencing participation, and build predictive models for better understanding of gender representation in legislative bodies.

🎯 **Objectives**
Analyze women's parliamentary participation data.
Perform data cleaning and preprocessing.
Conduct exploratory data analysis (EDA).
Visualize trends and patterns using graphs and plots.
Build a Decision Tree model for prediction and interpretation.
Extract meaningful insights from the dataset.
