# 👩‍⚖️ Women's Participation in Parliament Analysis using R

> A comprehensive data analytics project exploring global trends in women's parliamentary representation through statistical analysis, visualization, and machine learning.

## 🚀 Project Highlights

✅ Cleaned and analyzed parliamentary participation data

✅ Performed Exploratory Data Analysis (EDA) on socio-political indicators

✅ Generated multiple visualizations to identify trends and outliers

✅ Built and visualized a Decision Tree model for interpretable predictions

✅ Extracted actionable insights regarding gender representation in governance
